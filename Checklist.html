list-style-type屬性是用來設定在清單中，每一條之前的記號要用什麼

    list-style-type: none;(沒有)
    list-style-type: disc;(全黑圓圈)
    list-style-type: circle;(空心圓圈)
    list-style-type: square;(正方形)

    list-style-type:decimal-leading-zero;(阿拉伯數字/十進制前置零)
    list-style-type:decimal;(阿拉伯數字)
    list-style-type:upper-latin;(大寫拉丁文)
    list-style-type:lower-latin;(小寫拉丁文)
    list-style-type:upper-roman;(大寫羅馬文)
    list-style-type:lower-roman;(小寫羅馬文)
    list-style-type:upper-alpha;(大寫英文字母 )
  • aaa
  • bbb
  • ccc
    list-style-type:lower-alpha;(小寫英文字母 )
    list-style-type:lower-greek;(希臘文 )
    list-style-type:armenian;(亞美尼亞文 )
    list-style-type:georgian;(格魯吉亞語 )




list-style-position屬性讓我們可以決定記號是否應該被視為是文字的一部份， 並且由此而決定記號的位置





list-style-image屬性是用來將某個圖案設定為記號





list-style以上提到的這三個 (list-style-style、list-style-position、及 list-style-image) 屬性可以被簡化為一個 list-style屬性
